# Anime Production Workflow

## Consistency Gate

Before drawing, compare the requested variation with the character pack. Preserve head-to-body ratio, jaw geometry, eye spacing and highlight pattern, hair mass silhouette, signature items, and canonical palette. Record intentional deviations in `scene_notes`.

## Passes

1. Storyboard: frame, camera, eye line, silhouette readability, dialogue space.
2. Construction: head axis, rib cage/pelvis, joints, hand/foot direction, perspective anchors.
3. Lineart: stabilized strokes, deliberate line weight, closed contours where flats need them.
4. Flats: separate material/color regions; do not paint shadows into base colors.
5. Lighting: one explicit key-light direction, grouped shadow shapes, selective highlights.
6. Effects: atmosphere, speed lines, rain, glow, or depth cues on separate layers.
7. Review: zoomed-out silhouette, face consistency, anatomy, tangent collisions, panel flow, and layer names.

## Iteration

Export a preview after storyboard/rough and after lineart. Modify the smallest failing phase. Save numbered `.kra` files and keep the JSON plan and execution report beside them for replay.
